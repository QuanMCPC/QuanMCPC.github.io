How to make a website in Github?

Step 1. Create a Github account or log into your existing Github account

Step 2. Create a new repository by clicking on the "Create repository" button on the left or click the "+" on the near top-right corner then "New repository"

Step 3. IMPORTANT! In the repository name, you have to name that repository {your_username}.github.io (Replace the {your_username} with your username, example: example.github.io), make sure that the repo is public then press "Create repository"

Step 4. After that, create a new file and name that file "index.html" (without quotes). This will be the file that load into the browser when type the address in

Step 5. Add your HTML code into the file, type the address into the address bar and the website should show up
